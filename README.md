## General information

This repo provides examples of linear regression, data visualization, and statistical critique (the last of which using data from the following paper:

T. R. McClanahan and S. H. Shafir. 1990. "Causes and consequences of sea urchin abundance and diversity in Kenyan coral reef lagoons." _Oecologia_ 83, 362–370. doi: 10.1007/BF00317561
)

This is a fork of a repo from the ENVS-193DS-2026 organization.

## Data and file information

There is a `data` folder in which data files are stored.

There is a `code` folder in which all code for linear models are stored.
